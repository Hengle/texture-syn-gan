# Periodic Spatial GAN for Texture Synthesis

### Usage
```bash
cd src/dcgan
python main.py --dataroot $PATH_TO_DTD_DATA --config archpc
```

### Datasets
* [Describable Textures Dataset (DTD)](https://www.robots.ox.ac.uk/~vgg/data/dtd/)
